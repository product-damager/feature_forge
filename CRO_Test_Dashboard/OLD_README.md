# Antigravity Workshop Template 🚀

Welcome to your rapid prototyping starter kit! This folder contains everything you need to build and share interactive product ideas.

## Getting Started

### 1. Create a GitHub Account
If you don't have one, create a free account at [github.com](https://github.com).

### 2. Use this Template
- Click the **"Use this template"** button at the top of this repository (if available).
- **OR**: Create a new repository on GitHub and push these files to it.
- Open the new repository in VS Code/Antigravity.

### 3. Enable Live Deployment 🌍
1.  Go to **Settings** > **Pages**.
2.  Under **Build and deployment** > **Source**, select **GitHub Actions**.
3.  Go to the **Actions** tab to see your first deployment running. Once green, your prototype is live!

---

## 4. Launch the Workshop ⚡️

**Copy and paste the command below into Antigravity (Cmd+K / Ctrl+K):**

> **"Read the files in this directory and follow the instructions in WORKSHOP_START_HERE.md to initialize my workspace, task list, and implementation plan in one go."**

This will cause Antigravity to autonomously analyze your project and prepare a professional development environment for you.

---

### Tips for PMs
- **Collaborate**: Treat the agent as a senior coworker. Ask for their opinion on your ideas.
- **Gallery**: Use the `gallery_instructions.md` file if you want to turn this repo into a directory of multiple prototypes.
- **Documentation**: Check the `implementation_plan.md` and `decisions.md` files that the agent creates to see the roadmap and trade-offs.

Happy building!
